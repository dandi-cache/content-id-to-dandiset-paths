# DANDI Cache Logo

This document describes the DANDI Cache logo.

## Logo File

The logo is located at: `assets/logo.svg`

## Design Concept

The DANDI Cache logo combines the iconic DANDI Archive brain with cache-specific elements, creating a unified visual identity that clearly connects to DANDI while representing caching functionality. The logo is text-free for universal recognition.

### Core Elements

**DANDI Brain Icon** (Center)
- The recognizable brain from the DANDI Archive logo
- Pink/coral colors (#F0A5AC, #D3868D, #A05A60)
- Represents neuroscience data and the DANDI ecosystem
- Scaled and positioned as the central focal point

**Cache Network Ring** (Surrounding)
- Four arc segments forming a circular ring around the brain
- Teal color (#4A90A4) with transparency
- Represents distributed cache infrastructure
- Visual metaphor for data caching layers

**Cache Nodes** (4 cardinal points)
- Concentric circles at top, right, bottom, and left positions
- Light blue outer (#A8D5E2), teal middle (#4A90A4), dark blue center (#00436D)
- Represent cache access points and network nodes
- Connect visually to the surrounding ring

**Hash Symbols** (4 corners)
- Hash (#) symbols in each corner
- Cyan color (#6BB6CC) with transparency
- Represent key-value pairs and content addressing
- Core concept of cache lookup mechanisms

**Data Flow Arrows** (Cardinal directions)
- Small directional arrows between nodes and brain
- Cyan color (#6BB6CC) with transparency
- Indicate active data transfer to/from cache

## Visual Metaphor

The logo creates a cohesive representation of:
- **DANDI Identity**: Central brain maintains brand connection
- **Cache Infrastructure**: Ring and nodes show caching architecture
- **Content Addressing**: Hash symbols represent key-based lookups
- **Data Flow**: Arrows show active data movement
- **Distributed System**: Multiple nodes accessing central data

## Design Rationale

1. **DANDI Connection**: Uses official DANDI Archive brain icon maintaining brand recognition
2. **Text-Free**: Universal symbol transcending language barriers
3. **Cache-Focused**: Ring, nodes, and hash symbols clearly indicate caching functionality
4. **Hybrid Approach**: Balances DANDI identity with cache-specific elements
5. **Professional**: Clean, technical aesthetic appropriate for developer tools
6. **Scalable**: Square format and SVG ensure flexibility across use cases
7. **Complementary Colors**: Cache colors (teal/blue) complement brain colors (pink/coral)
